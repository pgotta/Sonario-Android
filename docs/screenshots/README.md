# Screenshots

Drop three PNG screenshots here with these exact filenames so the main README
displays them:

- `main.png`    - the main screen: input box, engine toggle, Summarize button.
- `summary.png` - a finished summary (e.g. the Bitcoin YouTube video with the
                  Normal / Detailed / Bullets tabs).
- `setup.png`   - the first-run setup screen (model picker + "Use Groq cloud").

## How to capture on your phone

1. Open Sonario and get to the screen you want.
2. Press Volume Down + Power together to take a screenshot.
3. Transfer the images to your computer (USB, Google Drive, or email).
4. Rename them to the names above and copy them into this folder.
5. Commit and push. GitHub will render them in the README automatically.

Tip: you already have suitable screenshots in your chat history - the summarized
Bitcoin video, the main input screen, and the setup screen all work well.

PNG or JPG both display; if you use JPG, change the extensions in the main
README's Screenshots section to match.
